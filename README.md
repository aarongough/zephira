# Zephira

Zephira is the ultimate AI-powered command-line coding assistant, written in Ruby. It delivers a clean, pipeline-driven agent loop, a rich function-calling toolkit, and an extensible plugin system to streamline your development workflow.

## Installation

`gem install zephira`

## Development

Clone this repo:
`git clone https://github.com/aarongough/zephira.git`

Install the development dependencies:
`bundle install`

Running specs locally:
`bundle exec rspec`

Running linter:
`bundle exec standardrb --fix`
